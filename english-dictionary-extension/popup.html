웹 페이지에서 영어 단어의 의미를 즉시 확인하세요

  • 단어 드래그/호버 시 즉시 사전 검색
  • 영어 의미 + 한글 번역 동시 표시
  • 영어 예문 + 한글 번역 제공
  • 발음 기호 및 음성 재생
  • 확장 프로그램에서 직접 단어 검색

📖 사용 방법

• 드래그: 궁금한 영어 단어를 드래그하세요

• 호버: 단어 위에 마우스를 올려두세요 (300ms 후 표시)

• 한글 뜻: 영어 의미와 한글 번역을 함께 표시

• 예문: 영어 예문과 한글 번역 제공

• 발음: 🔊 버튼으로 원어민 발음 재생

• 닫기: ESC 키나 팝업 외부 클릭으로 닫기
